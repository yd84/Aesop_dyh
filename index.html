WebGL Liquid Metal

把液态金属效果搬进你的展示页。

当前版本保留原项目的核心图形思路：WebGL fragment shader、Simplex Noise 扰动、SDF/Poisson 风格 mask，以及随时间变化的折射色散条纹。为了便于静态部署，我去掉了 Tweakpane 和视频导出，只保留主视觉渲染。

01

WebGL Shader

使用 GLSL 在浏览器端实时计算液态金属纹理、折射与透明边缘。

02

Swift Mask

以内嵌 SVG 生成形状 mask，再通过 Poisson 迭代形成适合 shader 采样的灰度场。

03

Fluid Motion

通过 Simplex Noise 和时间变量让纹理呈现持续流动，而不是普通发光或静态渐变。

04

Static Deploy

全部代码封装在 index.html，无需 Vite、npm 或后端服务，可直接部署为静态网页。

Core liquid-metal rendering technique adapted from the MIT-licensed open-source project eamonliu/liquid-metal-playground. This single-file adaptation removes GUI controls and video export, then integrates the WebGL visual into the current Vectoria page structure.
